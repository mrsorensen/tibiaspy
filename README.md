# tibiaspy
Python script that prints Tibia character info and online status
## How to run
```
python ~/tibiaspy/tibiaspy.py
```
## How it looks
```
Name:           Bubble    
Level:          273
Gender:         female
World:          Refugia
Vocation:       Knight
Residence:      Thais
Guild:          Amicitia Semper Prodest
Comment:        Nietzsche is my husband.
Status:         Offline
Last login:     Jul 30 2015, 08:03:11 CEST
```
